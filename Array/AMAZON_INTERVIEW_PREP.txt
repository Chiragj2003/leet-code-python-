Problems: #283 (Move Zeroes), #75 (Sort Colors), #56 (Merge Intervals)
When: Need to manipulate array from both ends or track boundaries

When: Dealing with pairs, XOR cancellations, or binary operations

PATTERN 3: DYNAMIC PROGRAMMING
Problems: #152 (Max Product), #238 (Product Except Self)
When: Need to track state from previous positions
Key: Track both positive/negative states or left/right passes

PATTERN 4: SORTING + GREEDY
Problems: #56 (Merge Intervals)
When: Need to organize data first, then process greedily
Key: O(n log n) for sorting often worth it

PATTERN 5: IN-PLACE HASHING
Problems: #41 (First Missing Positive)
When: Can use array indices as hash keys
Key: Clever use of array itself instead of extra hashmap

PATTERN 6: VOTING ALGORITHM
Problems: #169 (Majority Element)
When: Finding dominant element or majority patterns
Key: Cancel out minorities, majority survives

PATTERN 7: MATRIX MANIPULATION
Problems: #48 (Rotate Image)
When: Working with 2D arrays
Key: Transpose + reverse, or layer-by-layer approaches
